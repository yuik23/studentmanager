# 学生管理プログラム (初級理解度チェック 課題2)
## 概要
学生の名前と点数を管理するプログラムを作成しました。

１～６の番号を選ぶことによって以下の操作が可能です。
  1. 学生を追加
  2. 学生を削除
  3. 点数を更新
  4. 平均点を計算
  5. 全学生の情報を表示
  6. 終了
